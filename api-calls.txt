PROFILE API's:
1.@route:/addprofile
  @desc:POST request to add profile data
2.@route:/studentprofiledetails?rollNo={rollNo}
  @desc:GET request to fetch profile data
3.@route:/studentprofilepicture?rollNo={rollNo}
  @desc:GET request to fetch profile picture
4.@route:/updateprofile
  @desc:POST request to update profile data
5.@route:/removestudentprofiledetails?rollNo={rollNo}
  @desc:GET request to remove profile data

USER LOGIN API's:
1.@route:/createuser
  @desc:POST request to create user
2.@route:/validateuser
  @desc:POST request to validate user
3.@route:/user?username={username}
  @desc:PUT request to change login credentials
4.@route:/user?username={username}
  @desc:DELETE request to delete user
